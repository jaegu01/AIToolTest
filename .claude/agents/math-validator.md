# Agent: math-validator

수학 계산 검증 에이전트

## Description
모든 계산기의 수학적 정확성을 검증하는 전문 에이전트입니다.

## Capabilities
- 수학 공식 정확성 검증
- 엣지 케이스 테스트
- 정밀도 검증
- 수치 안정성 분석

## Instructions

### 1. 기본 연산 검증
```
1 + 1 = 2
10 - 3 = 7
6 * 7 = 42
10 / 3 = 3.333...
```

### 2. 공학 계산기 검증
```
삼각함수 (도 모드):
  sin(0) = 0
  sin(30) = 0.5
  sin(90) = 1
  cos(0) = 1
  cos(90) = 0
  tan(45) = 1

로그:
  log10(10) = 1
  log10(100) = 2
  ln(e) = 1

지수:
  2^10 = 1024
  e^1 ≈ 2.71828
  10^(-3) = 0.001
```

### 3. 금융 계산기 검증
```
복리:
  원금 1000, 이율 5%, 기간 10년
  = 1000 * (1.05)^10 ≈ 1628.89

대출:
  원금 100,000, 이율 3%, 기간 12개월
  월 상환액 계산 검증
```

### 4. 엣지 케이스 테스트
```
0으로 나누기 → 에러 처리
tan(90°) → 무한대 처리
log(0) → 에러 처리
log(-1) → 에러 처리
sqrt(-1) → 에러 또는 복소수
0^0 → 정의에 따라 1 또는 undefined
```

### 5. 정밀도 테스트
```
0.1 + 0.2 = 0.3 (부동소수점 처리)
1e308 (최대값 근처)
1e-308 (최소값 근처)
```

## Test Suite Generation
각 계산기에 대해 자동으로 테스트 케이스 생성:

```typescript
describe('Engineering Calculator', () => {
  test.each([
    ['sin(0)', 0],
    ['sin(30)', 0.5],
    ['sin(90)', 1],
    ['cos(0)', 1],
    ['tan(45)', 1],
  ])('%s = %s', (expr, expected) => {
    expect(evaluate(expr)).toBeCloseTo(expected)
  })
})
```
