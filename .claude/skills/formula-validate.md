# Skill: formula-validate

수학/과학 공식 검증 스킬

## Description
계산기에서 사용하는 수학 및 과학 공식의 정확성을 검증합니다.

## Instructions

1. 수학 공식 검증:
   - 삼각함수 정의 및 항등식
   - 로그 법칙
   - 지수 법칙
   - 행렬 연산 규칙

2. 물리 공식 검증:
   - 뉴턴 역학: F=ma, E=mc², 등
   - 전기: V=IR, P=IV, 등
   - 열역학: Q=mcΔT, 등

3. 화학 공식 검증:
   - 몰 계산: n=m/M
   - pH 계산: pH=-log[H+]
   - 농도 계산

4. 금융 공식 검증:
   - 복리: A=P(1+r/n)^(nt)
   - 현재가치/미래가치
   - 대출 상환 공식

5. 테스트 케이스 생성:
   - 경계값 테스트
   - 알려진 결과와 비교
   - 특수 케이스 (0, 음수, 무한대)

## Output Format
```
=== 공식 검증 결과 ===

[Engineering Calculator]
✓ sin(90°) = 1
✓ log10(100) = 2
✗ tan(90°) = 처리 필요 (무한대)

[Financial Calculator]
✓ 복리 계산 정확
✓ 대출 계산 정확

권장 수정사항:
1. tan(90°) 특수 케이스 처리 추가
```
